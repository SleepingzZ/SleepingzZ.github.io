变量与常量

Author: Celeste° Date: 2017-September-17  14:53:58 Category: Python

变量

什么是变量

  • 变 → 事物的状态是可以变化的
  • 量 → 事物的状态
  • 变量 → 一种语法机制, 可以把事物的状态储存起来, 并且可以取出来修改, 本质是一种存取内存的机制

如何使用变量

  • 变量的使用遵循先定义, 后引用原则

变量的三大组成部分

  • 变量名 → 访问变量值
  • 赋值符号 → 将比那辆值对应的内存地址绑定给变量名
  • 变量值 → 记录事物的状态, 存储数据

变量名的命名规则

  • 见名知义
  • 规范
    1. 字母、数字、下划线
    2. 不能以数字开头
    3. 不能与 Python 关键字冲突
  • 命名风格
    1. 纯小写加下划线(推荐)
    2. 驼峰体

变量值

  • 变量的量大特征
    1. id 号 → 变量值的身份证号, 反应的是内存地址, 但不是内存地址
    2. type → 数据类型

常量

什么是常量

  • 程序运行过程中，值不会发生变化的量

常量的命名

  • 在 Python 中，通常用全部大写的变量名表示常量 → PI = 3.14159265359

常量的本质

  • 但事实上，从 Python 语法角度看，PI 仍然是一个变量，因为 Python 根本没有任何机制保证 PI 不会被改变。你完全可以给 PI 赋值为 10，不会弹出任何错误。所以，用全部大写的变量名表示常量只是一个习惯上的用法。
  • 常量通常放置在代码的最上部，并作为全局使用。
