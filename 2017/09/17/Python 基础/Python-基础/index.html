Python 基础

Author: Celeste° Date: 2017-September-17  14:53:58 Category: Python

一、认识Python

  • 理解所有语法的大前提：创造语法是为了让计算机能够具备人的某项功能
  • 编写 Python 的两个地方：
    • 交互式环境（DOS命令）：主要用于辅助调试，多用于测试单行代码
    • 写入文件（主流：永久保存代码）

  • 运行 Python 程序的三个阶段

    1. 先运行 Python 解释器

    2. Python解释器将 a.py 的内容从硬盘读入内存

    3. Python解释器识别刚刚读入的内容，解释执行

      PS：我们的python程序调用的都是解释器的功能

1. 注释（PEP8：Ctrl + Alt + L，标准格式化）

  • 对代码进行解释说明

  1. 多行注释

    • 多行注释加在整个文件的开头，用来对该文件进行一个整体性的说明
    • 通常一个文件中只有一个多行注释

  2. 单行注释（Ctrl+/）

    • 主流注释
    • 一般添加在被解释代码的正上方

2. 变量

  1. 什么是变量？

    • 量：事物的状态

    • 变：事物的状态是可以变化的

    • 变量就是一种机制/语法：

      答：可以把事物的状态储存起来，并且以后可以取出来修改

  2. 为何要用变量？

    • 为了让计算机能够像人一样记住事物的状态，并且以后可以改变
    • 变量其实是一种存取内存的机制

  3. 如何用变量？

    • 原则：先定义，后引用

      • 定义语法：

        变量名 = 变量值

  4. 变量的三大组成部分

    • 变量名：访问变量值
    • 赋值符号：把变量值的内存地址绑定给变量名
    • 变量值：就是我们记录的事物的状态，即存储数据

  5. 变量名命名

    • 大前提：见名知意
    • 规范：
      • 字母、数字、下划线组成
      • 不能以数字开头
      • 不能与 Python 关键字冲突
    • 命名风格：
      • 纯小写加下划线（变量的命名推荐使用该种风格）
      • 驼峰体

  6. = 左边赋值，右边为取值，即左值为赋值，右值为取值

  7. 变量值

    • 变量有两大特征

      • id 号：是变量值的身份证号，反应的是内存地址，但不是内存地址
      • type 数据类型

    • 内存管理机制

      • 垃圾回收机制 GC：

        • 引用计数回收：计数判断为 0 即回收（del函数为解除关联，并非删除）
        • 分代回收：辅助回收，解决引用计数效率问题，对一部分长期使用变量降低扫描频率
        • 标记/清除：引用计数有可能发生内存泄漏
        • PS：如果想要回收Python程序的内存空间，只能通过把变量值的引用计数减为0的方式实现

      • 小整数池（在 DOS 命令下该范围为 -5，256，在 PyCharm 中这个范围被进一步优化）：

        • is 判断 id ，== 判断值
        • id 相等，值一定相等，即 x is y –> x == y
        • 值相等，id不一定相等，即 x == y != x is y
        • 判断是否为 None 用 is 连接，例 print(x is None)

  8. 语法错误所有代码不执行，逻辑错误先运行后报错

3. 基本数据类型

  1. 整型 int：记录人的年龄、等级、号码、个数

  2. 浮点数 float：记录人的薪资、身高、体重

    • int 和 float 可统称为数字类型

  3. 字符串 str：记录描述性质的状态，比如名字、语句、性别

    • 字符串之间可以相加 + ，或用 * 重复

  4. 列表类型：按照位置存放多个值，比如记录一个人的爱好。按清晰程度，一般用于存储同种属性的值

  5. 字典类型：一般用于存储不同属性的值，按照 key : value 的格式存放多个元素，其中 value 可以是任意类型，而 key 通常是 str 类型

  6. 布尔类型 bool：记录真假

    PS：

    • Python的数据类型之间边界十分明显，不能混用，且为动态类型，只有在程序运行后才能知道数据类型
    • 判断数据类型是否为 None，一般用 is
