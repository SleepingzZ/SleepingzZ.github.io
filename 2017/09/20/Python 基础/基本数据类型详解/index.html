  • 用途：名字，性别，国籍，地址等描述信息

三、列表类型

定义方式

  • 中括号括起来 内存可以存放多个元素，元素与元素之间逗号隔开 元素可以是任意数据类型
  • 能够支持for循环的数据 都能够被list转换成列表

1. 索引取值

2. 切片操作

3. 统计列表中元素的个数

四、字典类型

定义方式

  • key 必须是不可变类型，value 可以是任意类型

作用

  • 存多个值：key-value 存取，取值速度快

1. 按 key 存取值：可存可取

2. 长度len 统计字典中键值对的个数

3. 成员运算in和not in

4. 删除

5. 键 keys()，值 values()，键值对 items()

6. 循环

7. 取值

8. 创建字典

9. 补充

五、元组

定义方式

  • 小括号括起来 括号内可以存放多个元素，元素与元素之间逗号隔开 元素可以是任意类型，元组也可以简单的理解为是列表，元组属于不可变类型
  • 元组内元素的内存地址不能修改，但是如果元素是可变类型 那么元素值可以改

作用

  • 存多个值，对比列表来说，元组不可变（是可以当做字典的 key 的），主要是用来读

1. 类型转换

  • 类型转换与 list 一致，支持 for 循环的都可以

2. 按索引取值(正向取+反向取)：只能取

3. 切片(顾头不顾尾，步长)

4. 长度

5. 成员运算 in 和 not in

6. 循环

六、集合

定义方式

  • 大括号括起来 内部可以存放多个元素
  • 元素与元素之间逗号隔开(不是k:v键值对)，元素只能是不可变类型(整型、浮点型、字符串、元组)

使用场景

  1. 关系运算
    • 两个群体之间判断 交叉并集
    • 共同好友 共同关注 共同点赞…
  2. 去重操作
    • 去除重复的元素

1. 去重

2. 关系运算

七、可变不可变

  • 可变类型
    • 值改变，内存地址不变
  • 不可变类型
    • 值改变，内存地址一定变

不可变类型

  • Str, Int, Float, Tuple, Date, Bool, Set

可变类型

  • List, Dict
