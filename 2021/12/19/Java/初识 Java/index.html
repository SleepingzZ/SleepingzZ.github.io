初识 Java

一、Java 语言特性

  • 开源、免费、纯面向对象、跨平台性
  1. 简单性
    • 相对 C++ 而言，Java 中不在支持多继承，C++支持多继承，多继承较为复杂
    • C++ 中有指针，Java 屏蔽了指针的概念
    • Java 底层由 C++实现，但不是 C 语言
  2. 面向对象
    • Java 是纯面向对象的，更符合人的思维模式，更容易理解
  3. 跨平台性（可移植性）
    • Java 程序可以做到一次编译，到处运行 – 即编译之后不做任何修改即可在任意操作系统上运行
    • Java 程序运行于 Java 虚拟机之上，简称 JVM，再由 Java 虚拟机与底层操作系统记性交互
  4. 健壮性
    • 自动垃圾回收机制 – GC 机制
    • Java 语言运行过程中产生的垃圾是会被自动回收
  5. 多线程
  6. 安全性

二、Java 的加载与执行

  • Java 程序运行包含两个阶段
    1. 编译阶段
    2. 运行阶段

  1. 编译阶段

    • 检查 Java 源程序是否符合 Java 语法，符合 Java 语法才能够使用 javac.exe 进行编译，生成正常的字节码文件 xxx.class
    • 字节码文件不是纯粹的二进制，这种文件无法在操作系统当中直接执行
    • class 文件生成之后，删除 Java 源文件并不会影响 Java 程序的执行
    • 编译结束后，可以将 class 文件拷贝到其他操作系统运行

  2. 运行阶段

    • JDK 安装之后，除了自带一个 javac.exe 之外，还有另一个工具/命令，叫做 java.exe 命令，主要负责运行阶段
    • java.exe –> Java 虚拟机（JVM）–> JVM 启动类加载器 ClassLoader
    • ClassLoader 回去硬盘上搜索 xxx.class 文件，找到该字节码文件装载到 JVM 中
    • JVM 将 xxx.class 字节码文件解释成二进制数据
    • 最后由操作系统执行二进制和底层硬件平台进行交互
