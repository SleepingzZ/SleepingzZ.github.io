概述

什么是程序

  • 程序就是完成某个功能的指令的集合

特点

  • Go 语言保证了既能到达静态编译语言的安全和性能，又达到了动态语言开发维护的高效率，Go = C + Python，即说明 Go 语言既有 C 静态语言程序的运行速度，又能达到 Python 动态语言的快速开发
  • 从 C 语言中继承了包括表达式语法、控制结构、基础数据类型、调用参数传值、指针等等理念，也保留了和 C 语言一样的执行方式及弱化指针
  • 引入包的概念，用于组织程序结构，Go 的一个文件都要归属一个包，而不能单独存在

  • 垃圾回收机制，内存自动回收，不需要开发人员管理

  • 天然并发

    • 从语言层面支持并发
    • goroutine - 轻量级线程，可实现大并发处理，高效利用多核
    • 基于 CPS 并发模型（Communicating Sequential Processes）实现
    • 吸收了管道通信机制，形成 Go 语言特有的管道 channel – 通过管道 channel，可以实现不同的 goroute 之间的相互通信
    • 函数返回多个值
    • 新的创意，比如切片 slice、延时执行 defer 等

Go 开发注意事项

  1. Go 源文件以 ‘go’ 为扩展名
  2. Go 应用程序的执行入口是 main() 函数
  3. Go 语言严格区分大小写
  4. Go 方法有一条条语句构成，每个语句后不需要分号(Go 会在每行后自动添加分号)
  5. Go 编译器是一行行进行编译的，因此我们一行就写一条语句，不能把多条语句写在同一个
  6. Go 语言定义的变量或者 import 的包如果没有使用到，代码不能编译通过
  7. 大括号必须成对出现，缺一不可
